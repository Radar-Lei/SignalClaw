# Scenarios

This lightweight public repository does not bundle the full SUMO scenario assets.

The experiment scripts in `scripts/` expect scenario folders such as:

- `scenarios/arterial4x4_t1`
- `scenarios/arterial4x4_t2`
- `scenarios/arterial4x4_t3`
- `scenarios/emergency_e1`
- `scenarios/emergency_e2`
- `scenarios/transit_b1`
- `scenarios/transit_b2`
- `scenarios/incident_i1`
- `scenarios/mixed_m1`

Add the corresponding SUMO network and route files here before running end-to-end experiments.
